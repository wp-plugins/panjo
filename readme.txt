Embed a live feed of Panjo listings from your favorite marketplace. 

== Description ==

This plugin allows you to display a live feed of Panjo listings from popular marketplaces. Listings can be displayed either on a widget sidebar or on a whole page. If a desired marketplace is not included in the plugin settings, users may add their own marketplace.

Learn more about the Panjo marketplace here: http://www.panjo.com/faq

= Display options =

Number of listings to show

Selected marketplace

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Under the ‘Settings Menu’->’Panjo’ in your wordpress dashboard, select the desired number of listings.
4. Select the desired marketplace.
5. Save changes.

If you would like to add the widget to the sidebar, follow these steps.

1. Go to the ‘Appearance’ page in your wordpress dashboard.
2. Drag the Panjo widget under ‘Available Widgets’ into the ‘Widget Area’ area.


If you would like to add the widget to a page, follow these steps.

1. Under the ‘Settings Menu’->’Panjo’ in your wordpress dashboard, make sure you have selected the correct marketplace and saved your changes.
2. Copy the shortcode [panjopageview market=”*marketplace-name*”].
3. Paste this text into your pages content.


== Frequently Asked Questions ==
How can I get listings to display in a grid view?

At this time the only way to display listings in a grid view is to use the shortcode that our plugin's settings page will give you.
